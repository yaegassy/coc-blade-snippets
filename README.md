# coc-blade-snippets

## Usage

To use it, you need to install [coc-snippets](https://github.com/neoclide/coc-snippets).

And set `snippets.loadFromExtensions` to `true` in "coc-settings.json"

```jsonc
{
  // ...snip
  "snippets.loadFromExtensions": true,
  // ...snip
}
```

## snippetUpdate

```sh
yarn snippets
```
